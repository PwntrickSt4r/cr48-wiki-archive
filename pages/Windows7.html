Install Windows 7 on the Cr-48


  1. Install the Insyde H2O BIOS by following the instructions here

  2. Boot and immediately press → (F2) to enter BIOS setup. Confirm the following settings are correct in the BIOS:

    • Advanced > IDE Configuration > HDC Configured As <AHCI>
    • Power > Advanced CPU Control > HT Support <Auto>
    • Boot > ACPI Selection <Acpi 3.0> (4.0 works also, but the BIOS says use 1.0 or 3.0)

  3. Obtain a copy of Windows 7. Both 32 and 64-bit versions will work. After installation you may have limited usable disk space unless you install a larger SSD.

  4. On another computer, create a bootable Windows 7 installation USB drive or SD card using the Windows 7 USB DVD Tool available here

  5. Insert the USB drive or SD card in the Cr-48 and boot to the Windows 7 installer. During the installation:

    • Select "Custom Install" rather than "Upgrade"
    • Select "Drive options (advanced)" and delete every partition on the internal Solid-State Drive (SSD)
    • Install Windows to 'Disk 0 Unallocated Space'

  6. After Windows finishes installing, remove the installation media and reboot. The reboot may happen automatically - be sure to remove the installation media to boot properly from the SSD - don't restart the installation process.

  7. Download the following drivers and install in this order:

    1. Intel Rapid Speed Technology
    2. Intel Chipset Installation Utility
    3. Intel Graphics Media Accelerator 3150
    4. Synaptics TouchPad Drivers
    5. Atheros Bluetooth

  8. Configure the keyboard by following the instructions here

  9. Configure the Gobi 2000 3G modem by following the instructions here
