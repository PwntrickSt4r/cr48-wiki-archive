Fix Touchpad in Ubuntu on the Cr-48


Follow these steps to enable two-finger scrolling and other multitouch features of the touchpad in Ubuntu 11.04.


Recent updates to Ubuntu appear to have resolved the touchpad issues on the Cr-48. Follow this guide only if problems with the touchpad persist after installing all normal updates with Update Manager.



The easy way:

  1. Install Ubuntu 11.04 on your Cr-48 by following Jay Lee's guide here

  2. Download the touchpad drivers here

  3. Double-click the .deb package you downloaded in step 2. This will launch Ubuntu Software Center.

  4. Click the 'Reinstall' or 'Upgrade' button (the button you'll see is based on which driver version is already installed)

  5. Click 'Install Anyway' if a warning appears about removing the x.org x server

  6. Enter your password to authenticate (the default password is 'user')

  7. Wait for the installation to complete and restart your Cr-48

  8. Adjust settings as desired in the Touchpad tab of Mouse Preferences.


If the easy way doesn't work...

You can compile your own drivers and install them by following these steps:

  1. Install Ubuntu 11.04 on your Cr-48 by following Jay Lee's guide here

  2. Boot into Ubuntu and open the Terminal

  3. Type mkdir $HOME/Desktop/build and press enter

  4. Type cd $HOME/Desktop/build && wget http://david.hardeman.nu/synaptics-suse-patches.tar.bz2 and press enter

  5. Type apt-get source xserver-xorg-input-synaptics and press enter

  6. Type cd xserver-xorg-input-synaptics* and press enter

  7. Type cd debian and press enter

  8. Type cd patches and press enter

  9. Type tar xfvj ../../../synaptics-suse-patches.tar.bz2 and press enter

  10. Type ls -1 2*.patch >> series and press enter

  11. Type cd ../.. and press enter

  12. Type sudo apt-get build-dep xserver-xorg-input-synaptics and press enter

  13. Type dpkg-buildpackage -us -uc -rfakeroot and press enter

  14. Type sudo dpkg -i ../xserver-xorg-input-synaptics_1*.deb and press enter

  15. Type sudo dpkg -i ../xserver-xorg-input-synaptics-dev* and press enter

  16. Restart your Cr-48 and adjust settings as desired in the Touchpad tab of the Mouse Preferences. Feel free to delete the folder entitled 'build' that is now on your desktop.


Source: https://bugs.launchpad.net/ubuntu-release-notes/+bug/582809/comments/144
