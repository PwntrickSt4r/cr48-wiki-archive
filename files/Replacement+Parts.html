Cr-48 Replacement Parts and Accessories

The following parts are confirmed to be compatible with the Cr-48:

LCD Panel

  • Original LCD panel is Samsung model LTN121AT10

AC Adapters

  • Original AC adapter is manufactured by Lite On Technology (Changzhou) Co., Ltd.
    • Model: PA-1400-20GI
    • Input: 100-240V ~1.0A 50-60Hz
    • Output: 19.5V 2.05A
  • Hewlett-Packard NA374AA#ABA Mini 30W AC Adapter
  • Lenmar 50-Watt Notebook Power Adapter Lac50 (Adjustable output voltage: 9.5V to 20V. Set to 19.5V for Cr-48)
  • FINsix DART Adapter ('A' tip w/4.0x1.7mm DC Male to 5.5x2.1mm DC Female HP adapter OR 'G' tip)

Battery

  • Currently no known source for compatible replacement battery
  • Original 63Wh lithium-ion battery manufactured by Hewlett-Packard
  • Specifications:
    • rechargeable: yes
    • energy-empty: 0 Wh
    • energy-full: 62.6632 Wh
    • energy-full-design: 62.5152 Wh
    • energy-rate: 9.4868 W
    • voltage: 15.255 V
    • technology: lithium-ion
  • Battery teardown photo 1
  • Battery teardown photo 2

Solid-State Drive (SSD)

  • Original SSD is SanDisk model SDSA4DH-016G 16GB mSATA SSD
  • Samsung 850 EVO 120GB - 1TB mSATA SSD
  • Crucial M550 128GB - 512GB mSATA SSD
  • MyDigitalSSD 32GB - 256GB mSATA SSD

WiFi

  • Original is Atheros Half Mini PCI-E Wireless Card model AR9280 AR5BHB92
  • Generally available on eBay

USB Ethernet Adapters

  • Agiler USB 2.0 Ethernet 10/100 Network LAN RJ45 Adapter
  • Cubeternet USB 2.0 Hi-Speed Fast Ethernet Adapter
  • Sabrent NT-USB20 USB 2.0 to RJ45 Fast Ethernet 10/100 Base-T Network Adapter
  • Rocketfish Network Adapter
  • D-Link DUB-E100 USB 2.0 Fast Ethernet Adapter
  • Nu USB 2.0 3-Port Hub with Ethernet Adapter
  • Cisco-Linksys USB300M
  • AX88178 - USB2.0 to 10/100/1000M Gigabit Ethernet Adapter

USB Hubs

  • IOGEAR 4-Port USB 2.0 Hub model GUH285
  • Belkin USB 2.0 4-Port Ultra Mini Hub model F5U407
  • Targus 4-Port USB 2.0 Ultra Mini Hub model PA055U

Audio Port Adapters

  • PC Headset to 3.5mm Smartphone Adapter - allows connection of smartphone headsets (microphone and headphone) to the headphone (TRRS) jack of the Cr-48

Wireless Mice and Keyboards

  • Any wireless mouse or keyboard with a USB dongle should work fine with the Cr-48
  • Set the speed of the mouse using the Touchpad settings at chrome://settings/system
  • Known-compatible wireless mice:
    • Microsoft Standard Wireless Optical Mouse 1000
    • Microsoft Wireless Notebook Optical Mouse 3000
    • Microsoft Wireless Mobile Mouse 4000
    • Logitech V450 Nano Cordless Laser Mouse
    • Logitech Wireless Mouse M305
    • Rocketfish Nano Laptop Laser Mouse
    • Gear Head 3 Button Wireless Optical Wheel Mouse model MP2350BLK
    • DAYTONA II, SwissGear Wireless Optical Mobile Mouse
