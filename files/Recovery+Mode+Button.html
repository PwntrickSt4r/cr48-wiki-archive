Cr-48 Recovery Mode Button


Recovery Mode is a boot operation where the BIOS will prompt you to insert a recovery USB drive.

The system will automatically enter Recovery Mode if the BIOS is unable to find a valid kernel to boot from.

If you are in Developer Mode you can enter Recovery Mode by pressing the space bar at the Chrome OS verification warning screen at startup.

If you are not in Developer Mode and want to enter Recovery Mode, you can use the Recovery Mode button.

  1. Shut down the computer and turn it over

  2. Directly below the right rear rubber foot is a small pinhole (see photograph below)

  3. Place a paperclip in the hole and press the button

  4. While pressing the button, turn the computer on

  5. You will now see the recovery screen and can proceed with the Chrome OS recovery process described here


Source: http://www.chromium.org/chromium-os/developer-information-for-chrome-os-devices/cr-48-chrome-notebook-developer-information
